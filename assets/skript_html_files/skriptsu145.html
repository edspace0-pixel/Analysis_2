3.12.1  Unbestimmtheiten vom Typ 0∕0.

Wir betrachten im weiteren Grenzwerte vom Typ lim x→x0f(x)∕g(x) wobei f(x0) = g(x0) = 0.

Theorem 3.12.1. Die Funktionen f :]a,b[→ Kn und g :]a,b[→ K mit K = {ℝ, ℂ} seien in einem Punkt x0 ∈]a,b[ differenzierbar und es gelte

f(x0) = g(x0) = 0sowieg′(x 0)≠0.

Dann ist der Quotient f(x) g(x) in einer ε-Umgebung des Punktes x0 wohldefiniert und es existiert der Grenzwert

lim x→x0f(x) g(x) = f′(x 0) g′(x0). (3.48)

Beweis. Aus der Differenzierbarkeit von g im Punkt x0 folgt

g(x) = g′(x 0)(x − x0) + o(x − x0)fürx → x0,

damit gilt g(x)≠0 für x ∈ Uε(x0) \{x0} für ein geeignetes ε > 0. Folglich ist f(x)∕g(x) in Uε(x0) \{x0} definiert. Desweiteren gilt

f(x) g(x) = f′(x 0)(x − x0) + o(x − x0) g′(x0)(x − x0) + o(x − x0)fürx → x0.

Jede Funktion ψ = o(x − x0) lässt sich als Produkt ψ(x) = (x − x0)ψ̃(x) mit ψ̃ = o(1) darstellen, woraus nach Kürzen mit x − x0

f(x) g(x) = f′(x 0) + o(1) g′(x0) + o(1)fürx → x0

folgt. Da g′(x 0)≠0, so erhält man daraus

f(x) g(x) = f′(x 0) g′(x0) + o(1)fürx → x0.

Letzteres impliziert (3.48). □

Problem 3.12.2. Führen Sie diesen Beweis in der ε-δ-Notation!

Theorem 3.12.3. Die Funktionen f :]a,b[→ Kn und g :]a,b[→ K mit K = {ℝ, ℂ} seien in einem Punkt x0 ∈]a,b[ m-fach differenzierbar und es gelte

f(k)(x 0) = g(k)(x 0) = 0,k = 1,…m − 1,sowieg(m)(x 0)≠0.

Dann ist der Quotient f(x) g(x) in einer ε-Umgebung des Punktes x0 wohldefiniert und es existiert der Grenzwert

lim x→x0f(x) g(x) = f(m)(x 0) g(m)(x0). (3.49)

Problem 3.12.4. Beweisen Sie diese Aussage nach dem Muster des Beweises von Satz 3.12.1.

Example 3.12.5. Wir berechnen den Grenzwert lim x→x0=0 sin x−x x3 . Für g(x) = x3 gilt g′(x) = 3x2, g′′(x) = 6x und g′′′(x) = 6 und damit

g(0) = 0,g′(0) = 0,g′′(0) = 0g′′′(0) = 6.

Für f(x) = sin x − x erhält man f′(x) = cos x − 1, f′′(x) = − sin x, f′′′(x) = − cos x und entsprechend

Damit ergibt sich

lim x→x0=0 sin x − x x3 = − cos(0) 6 = −1 6.

Wir formulieren nun die bekannte Regel von Bernoulli und l’Hospital.

Theorem 3.12.6. Die Funktionen f :]a,b[→ ℝn und g :]a,b[→ ℝ seien in ]a,b[ differenzierbar. Es gelte

lim x→af(x) = 0, lim x→ag(x) = 0sowieg′(x)≠0fürx ∈]a,b[.

Desweiteren existiere der Grenzwert

lim x→af′(x) g′(x) = A,A ∈ ℝ.

Dann existiert ebenfalls der Grenzwert

lim x→af(x) g(x) = A.

Beweis. Wir betrachten zunächst den Fall n = 1. Die Funktionen f und g können durch die Definition f(a) = 0 und g(a) = 0 stetig auf [a,x] für x ∈]a,b[ fortgesetzt werden. Nach der Formel von Cauchy gilt dabei

f(x) g(x) = f(x) − f(a) g(x) − g(a) = f′(c x) g′(cx)

für geeignetes cx ∈]a,x[. Strebt x gegen a, so konvergiert auch cx gegen a. Nach Voraussetzung existiert dann der Grenzwert

A = lim c→af′(c) g′(c) = lim x→af′(c x) g′(cx) = lim x→af(x) g(x).

Für vektorwertige Funktionen f wendet man dieses Argument individuell auf jede einzelne Komponente dieser Funktion an. □

Problem 3.12.7. Warum muss man den obigen Beweis für die einzelnen Komponenten einer Vektorfunktion f individuell führen?

Die Regel von Bernoulli und l’Hospital lässt sich nicht auf Brüche mit komplexwertigen Funktionen g im Nenner verallgemeinern. Formal scheitert der Beweis daran, dass die Formel von Cauchy nicht für komplexwertige Funktionen gilt. Folgendes Gegenbeispiel soll dies zusätzlich illustrieren:

Example 3.12.8. Es sei f(x) = x und g(x) = x + x2e i x2 mit x > 0. Man berechnet die Ableitungen von f und g als f′(x) = 1 und g′(x) = 1 + 2(x − i x)ei∕x2. Nach der Dreiecksungleichung gilt

|g′(x)|≥2(x − i x)ei∕x2 −|1| = 2x2 + 1 x2 − 1 ≥ 2 x − 1 = 2 − x x .

Daraus folgt f′(x) g′(x) ≤ x 2−x und A = lim x→0f(x) g(x) = 0, während auf der anderen Seite offensichtlich

lim x→0f(x) g(x) = lim x→0 x x + x2ei∕x2 = lim x→0 1 1 + xei∕x2 ︸ o(1) = 1≠A

gilt.

Problem 3.12.9. Kann man die Regel von Bernoulli und l’Hospital auf komplexe Vektorfunktionen f bei einer reellen Nennerfunktion g verallgemeinern?

Man kann die Regel von Bernoulli und l’Hospital auch für Grenzwerte x → +∞ anwenden. Dazu betrachten wir die differenzierbaren Funktionen f,g : ]b, +∞[ → ℝ, b > 0, welche durch die Substitution x = t−1 in die ebenfalls differenzierbaren Funktionen f̃,g̃ : ]0,b−1[ → ℝ mit f̃(t) = f(t−1), g̃(t) = g(t−1) übergehen. Die Ableitungen dieser Funktionen stehen in folgendem Zusammenhang

f̃t′(t) = −t−2f x′(x),g̃ t′ = −t−2f x′(x),x ∈]b, +∞[,t = x−1.

Gilt also g′(x)≠0 für x ∈]b, +∞[, so folgt g̃t′(t)≠0 für t ∈]0,b−1[. Es gelte

A = lim x→+∞f′(x) g′(x) = lim t→0 − t−2f̃(t) − t−2g̃(t) = lim t→0f̃(t) g̃(t).

Dann können wir Satz 3.12.6 anwenden und erhalten

lim x→+∞f(x) g(x) = lim t→0f̃(t) g̃(t) = A.
