4.1.6  Das Lebesgue-Kriterium der Riemann-Integrierbarkeit einer Funktion.

Wir formulieren hier ein weiteres hinreichendes und notwendiges Kriterium der Riemann-Integrierbarkeit. Für ein Intervall I ⊂ ℝ bezeichne |I| die Länge des Intervalls I.

Definition 4.1.13. Wir sagen, dass eine Menge E ⊂ ℝ das Lebesgue-Mass 0 besitzt, genau dann wenn für jedes ε > 0 eine höchstens abzählbare Familie von Intervallen {Ikε} k∈ℕ existiert, so dass

E ⊂⋃ k∈ℕIkεund∀ n∈ℕ ∑ k=1N|I k| < ε.

 

Definition 4.1.14. Wir sagen, dass eine Aussageform H fast überall auf [a,b] gilt, genau dann wenn eine Menge E ⊂ [a,b] mit dem Lebesgue-Mass 0 existiert, so dass

∀x∈[a,b]\EH(x)

wahr ist.

Wir geben den folgenden Satz von Lebesgue ohne Beweis an:

Theorem 4.1.15. Für eine Funktion f : [a,b] → ℝ gilt f ∈ R[a,b] genau dann, wenn f beschränkt ist und wenn f fast berall auf [a,b] stetig ist.
