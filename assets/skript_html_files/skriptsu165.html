4.4.2  Schriftliches Dividieren von Polynomen.

Polynome kann man schriftlich auf ähnliche Weise dividieren wie ganze Zahlen. Das folgende Schema soll dies illustrieren:

(x3+2x2+3x+1)÷(x+1) =x2 + x + 2 + 1 x+1 x3 + x2 x2 +3x x2 + x 2x+ 1 2x+2 1

In diesem Beispiel bleibt bei der Division der Rest 1∕(1 + x).
