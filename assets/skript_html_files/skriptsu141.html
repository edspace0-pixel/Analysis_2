3.10.3  Zu den Extremwerten einer Funktion.

Für eine stetige Funktion f : [a,b] → ℝ gibt es nach dem Satz von Weierstrass immer einen Punkt c ∈ [a,b] mit f(c) = max x∈[a,b]f(x). Letzteres ist äquivalent zur Aussage f(c) − f(x) ≥ 0 für alle x ∈ [a,b]. Der Satz von Fermat (Satz 3.5.1) formuliert mit f′(c) = 0 eine notwendige Bedingung dafür, dass das Maximum einer in ]a,b[ differenzierbaren Funktion in einem inneren Punkt c ∈]a,b[ angenommen wird. Wir diskutieren im weiteren hinreichende Bedingungen für die Existenz eines Extremwertes.

Theorem 3.10.4. Die Funktion f ∈ C([a,b], ℝ) sei differenzierbar in ]a,b[. Für den Punkt c ∈]a,b[ gelte f′(c) = 0 und zudem sei f′(x) ≥ 0 für x ∈]a,c[ sowie f′(x) ≤ 0 für x ∈]c,b[. Dann gilt

f(c) = max x∈[a,b]f(x).

Beweis. Wir betrachten einen Punkt x ∈ [a,c]. Nach dem Satz von Lagrange gilt

f(c) − f(x) = f′(ξ)(c − x)für  ein  geeignetesξ ∈]x,c[.

Nach Voraussetzung sind beide Faktoren auf der rechten Seite dieser Gleichung nichtnegativ und damit f(c) − f(x) ≥ 0 für alle x ∈ [a,c]. Den Fall x ∈ [c,b] löst man analog. □

Theorem 3.10.6. Die Funktion f ∈ C([a,b], ℝ) sei zweifach differenzierbar in ]a,b[. Für den Punkt c ∈]a,b[ gelte f′(c) = 0 und zudem sei f′′(x) ≤ 0 für alle x ∈]a,b[. Dann gilt

f(c) = max x∈[a,b]f(x).

Beweis. Aus f″(x) ≤ 0 folgt f′ ↓. Ist nun f′(c) = 0 so gilt damit f′(x) ≥ 0 für x ∈]a,c[ sowie f′(x) ≤ 0 für x ∈]c,b[. Wir können damit Satz 3.10.4 anwenden, was die geforderte Aussage sofort beweist. □

Definition 3.10.7. Man sagt die Funktion f : [a,b] → ℝ nimmt im Punkt c ∈]a,b[ ein lokales Maximum an, genau dann wenn Werte α,β ∈ [a,b], α < c < β existieren, so dass

f(c) = max x∈]α,β[f(x).

Theorem 3.10.8. Die Funktion f : [a,b] → ℝ sei im Punkt c ∈]a,b[ zweifach differenzierbar und es gelte f′(c) = 0 sowie f′′(c) < 0. Dann nimmt f im Punkt c ein lokales Maximum an.

Beweis. Nach dem Satz von Taylor gilt

f(x) = f(c) + f′(c)(x − c)︸ =0 + 1 2f″(c)(x − c)2 + r 2(x − c)

mit r2(x − c) = o(x − c)2 für x → c. Letztere bedeutet, dass für beliebiges ε > 0 ein δ > 0 existiert, so dass

|r2((x − c))|≤ ε(x − c)2,x ∈]c − δ,c + δ[.

f(x) − f(c) ≤−1 4|f′′(c)|(x − c)2 ≤ 0fürx ∈]c − δ,c + δ[,

was die Aussage beweist. □

Theorem 3.10.9. Die Funktion f : [a,b] → ℝ sei im Punkt c ∈]a,b[ n + 1-fach differenzierbar und es gelte

f(1)(c) = … = f(n)(c) = 0undf(n+1)(c) < 0.

Ist n gerade, so nimmt f in c weder ein Minimum noch ein Maximum an. Ist n ungerade, so nimmt f im Punkt c ein lokales Maximum an.
