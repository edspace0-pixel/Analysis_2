4.4.5  Die Formel von Ostrogradskij.

Aus Tl(x) Qn(x) = ∑ j=1N ∑ r=1kj Ajr (x − aj)r = ∑ j=1N Aj1 (x − aj) + ∑ j=1N ∑ r=2kj Ajr (x − aj)r (4.27)

folgt

∫ Tl(x) Qn(x)dx = ∑ j=1NA j1 ln |x − aj| + ∑ j=1N ∑ r=2kj Ajr (1 − r)(x − aj)r−1. (4.28)

Die zweite Summe auf der rechten Seite von (4.28) ist eine rationale Funktion. Für einen gegebenen Nenner

Qn(x) = bn(x − a1)k1 …(x − aN)kN

setzen wir nun Q̃N(x) = bn(x − a1)…(x − aN), Q^n−N(x) = (x − a1)k1−1…(x − a N)kN−1.

Dann folgt aus (4.27) und (4.28) die Darstellung

∫ Tl(x) Qn(x)dx = ∫ T̃r(x) Q̃N(x)dx + T^q(x) Q^n−N(x) (4.29)

mit geeigneten Polynomen T̃r(x) und T^q(x), wobei

deg(T̃r) = r ≤ N − 1, deg(T^q) = q ≤ n − N − 1.

Nach Differentiation erhält man

Tl(x) Qn(x) = T̃r(x) Q̃N(x) + T^q(x) Q^n−N(x) ′. (4.30)

Die Polynome T̃r(x) und T^q(x) erhält man aus (4.30) durch Koeffizientenvergleich.

Die Darstellungen (4.29) und (4.30) nennt man die Formel von Ostrogradskij. Diese spielt besonders bei der Integration von rationalen Funktionen mit Nennern mit Nullstellen höherer Ordnung eine Rolle. Dabei ist in der Anwendung besonders wichtig, dass Q^n−N(x) der grösste gemeinsame polynomiale Teiler von Q^N(x) und Q^N′(x) ist. Als solcher kann Q^n−N(x) (und damit auch Q^N(x)) mit dem Eulerschen Verfahren bestimmt werden, ohne dass vorher die Nullstellen von Qn(x) berechnet werden müssen!

Für deren Anwendung ist es insbesondere interessant, dass die Polynome Q^N(x) und Q^n−N(x) angegeben werden können, ohne dass man die explizite Nullstellenzerlegung von Qn(x) kennen muss. Für Einzelheiten verweisen wir auf Fichtenholz Bd. 2, Punkt 276.
