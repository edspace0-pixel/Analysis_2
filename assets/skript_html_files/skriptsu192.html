4.11.2  Der Beweis von Lemma 4.11.1.

Infolge der Ungleichung

|u + v|≥|u|−|v|

gilt

|f(x)|≥ S(x) − T(x),x ∈ ℂ,

mit S(x) = |anxn| = |a n||x|n, T(x) = |an−1xn−1 + … + a 0|.

Setzt man A = max k=0,…,n−1|ak|, so folgt

T(x) ≤ A(|x|n−1 + ⋯ + x + 1),x ∈ ℂ.

Insbesondere gilt für |x|≥ 1

T(x) ≤ nA|x|n−1,x ∈ ℂ,|x|≥ 1,

und damit S(x) − T(x) ≥ |an||x|n − nA|x|n−1 ≥ |x|n−1(|a n||x|− nA),x ∈ ℂ,|x|≥ 1.

Für gegebenes M > 0 setzen wir nun

N := max 1, M + n ⋅ A |an| .

Damit gilt für |x|≥ N sowohl |x|n−1 ≥ 1 als auch |an||x|− nA ≥ M und damit

|f(x)|≥ S(x) − T(x) ≥ M,x ∈ ℂ,|x|≥ N.
