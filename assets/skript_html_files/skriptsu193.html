4.11.3  Der Beweis von Lemma 4.11.2.

Zerlegt man das Polynom f vom Grad deg(f) = n im Punkt x∗ in eine Taylorreihe, so erhält man

f(x∗ + h) = f(x∗) + f′(x∗)h + … + f(n)(x∗)hn h! . (4.86)

Tatschlich, da die Ableitung f(n+1) verschwindet, so ist nach (4.44) der Restterm rn(x∗; h) = 0. Da an = (n!)−1f(n)(x)≠0 für alle x ∈ ℂ, so gibt es ein k ∈{1,…,n} mit der Eigenschaft
